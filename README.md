# Text Analysis Program
###### Python C++ 기반 텍스트 분석 프로그램

## Features

### 완료
- 단어 빈도 분석 (2025.01.26)

### 개발 예정
- 문장부호 제거
- 형태소 분석 (khaiii)
- 의미연결망 단어 매트릭스
- Word2Vec

## Requirements
- Ubuntu 24.04.1 LTS
- Python 3.9
- C++14

## Installation
```bash
git clone https://github.com/JaehnK/TextAnalyzer
cd TextAnalyzer
pip install -r requirements.txt
```

## Usage
- 실행: `word-freq`
- Docker 미지원
- Ubuntu 24.04.1 LTS에서만 실행 가능
